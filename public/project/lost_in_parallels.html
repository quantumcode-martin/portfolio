The Godot Fire Jam

Lost In Parallels is a 3 weeks game project made with Godot for the Godot Fire Charity Jam. Lost In Parallels won the first place, received a lot of positive feedbacks and a cashprice of $568.

The theme of the jam was "Everything Changes". Lost In Parallels is a multiverse platformer in which the player can explroe parallel universes using a flashlight. The world representation and physics are changin in real time depending on the flashlight position.

A Level Editor

After the game jam's results I felt the game had a lot of potential so I decided to create a level editor. This level editor was needed because the levels are interdependent and it makes the process of creating and modifying levels tiresome and time consuming.

As part of this level editor, I created a quite complex algorithm to compute the collision polygons for each layers and levels. This algorithm took me a full week of work to finish as a lot of spcific cases were to be taken into consideration.
