Introduction

IdleKiller is an idle game made with Godot Engine. It puts you in the shoes of a citizen living under a tyrannical government. Your job is to elimiate people. But you will soon discover that you may be able to change the course of events

This game was made in collaboration with a friend and collegue. It is currently in closed alpha testing on Google Play Store. We have around 50 testors and we have collected some user data and feedbacks to improve the game and make it ready for release.

The alpha version was made in only 3 mounths, and we made a resource oriented system so we can add and modify content very easely. This system could even be used to create other idle games super fast.
